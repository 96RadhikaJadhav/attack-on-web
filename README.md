
# Attack-on-Web(AoW)

## Test using CLI

   ```sh
  npm i atow
  npx atow
  ```
  
   [npx](https://nodejs.dev/learn/the-npx-nodejs-package-runner) is a package runner tool that comes with npm 5.2+ and higher.
